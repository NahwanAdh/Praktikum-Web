Pemrograman Web 1

Sesi 1

Praktik Pemrograman Web 1 akan membahas tentang cara menampilkan halaman web dengan menggunakan HTML, CSS, dan Javacsript. Mahasiswa diharapkan mampu memahami sintaks dasar dalam menampilkan halaman web, dan dengan pemahaman tersebut menuliskan hasil kreasinya sendiri.

  • HTML
  • CSS
  • Javascript

Pada sesi 1 mahasiswa mengenali dulu struktur dasar HTML beserta sintaks dasarnya.

  1. Pengenalan HTML
  2. Tabel
  3. Form
