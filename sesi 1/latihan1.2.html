Apa itu HTML?

HTML adalah bahasa standar untuk membuat halaman web.

  • Tentang HTML
  • Tentang CSS
  • Tentang JavaScript

HTML

HTML digunakan untuk membuat struktur halaman seperti heading, paragraf, dan list.

CSS

CSS digunakan untuk menghias tampilan halaman web agar lebih menarik.

JavaScript

JavaScript digunakan untuk membuat halaman web menjadi interaktif.
